# Bulgunniakhs Detection

Проект разработан в рамках ГИС-хакатона **GIS'IT 2026** как решение кейса №4 **«Выявление фактов деградации многолетней мерзлоты на землях сельскохозяйственного назначения»**.

## 📌 Описание

Программа автоматизирует выявление термокарстовых форм рельефа (булгунняхов) на спутниковых снимках высокого разрешения (Канопус-В) на землях сельскохозяйственного назначения. Анализ проводится для двух ключевых участков Республики Саха (Якутия):

- **Амга** (Amga)
- **Юнкор** (Yunkor)

Для каждого участка определены две зоны интереса (AOI):

- **`*_big_AOI`** – обрезка для сельскохозяйственных угодий (территория обработки полей);
- **`*_scene_AOI`** – экстент снимков Канопуса-В, охватывающий исследуемую территорию.

Алгоритм включает:
- предобработку спутниковых данных (приведение всех данных к 10-метровому разрешению);
- выделение потенциальных булгуннях/термокарстов/аласов на основе текстурных и спектральных признаков;
- оценку изменений во времени (тренды);
- визуализацию результатов в виде векторных слоёв.

## 🗂 Структура репозитория
bulgunniakhs-detection/  
├── GISIT_Якутск_Данные/ # папка со спутниковыми снимками Канопус-В (формат GeoTIFF).  
├── Земельный Кадастр_Якутия/ # папка с кадастровыми данными земель сельскохозяйственного назначения (формат CSV).  
│  
├── scene/ # директория содержащая AOI полигоны  
│ ├── Amga_big_AOI.geojson # экстент полигона сельхозугодий (участок Амга)  
│ ├── Amga_scene_AOI.geojson # экстент сцены Канопуса (Амга)  
│ ├── Yunkor_big_AOI.geojson # экстент полигона сельхозугодий (участок Юнкор)  
│ └── Yunkor_scene_AOI.geojson # экстент сцены Канопуса (Юнкор)  
│  
├── result/ # папка с результатами  
│ ├── composites/ # директория с композитами по территориям Amga/ & Yunkor/     
│ ├── yunkor_clipped.geojson # извлеченные по маске Yunkor_big_AOI С/Х угодья  
│ └── amga_clipped.geojson # извлеченные по маске Amga_big_AOI С/Х угодья  
│  
├── preprocessing.py # построение композитов и общая предобработка данных  
├── classification_model.py # классификация на основе UNET и Res-Net  
├── mask_agregation.py # агрегация бинарных масок для создания вектора оценки  
├── function.py # вспомогательные функции (обработка, визуализация)  
│  
├── README.md # этот файл  
└── LICENSE # лицензия MIT

## ⚙️ Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/your-username/bulgunniakhs-detection.git
cd bulgunniakhs-detection
```   
### 2. Установка зависимостей
Создайте виртуальное окружение (рекомендуется) и установите библиотеки из requirements.txt:   
```bash
pip install -r requirements.txt
```
К основным зависимостям относятся:
*  numpy~=2.2.6
*  rasterio~=1.4.3  
*  geopandas~=1.0.1  
*  affine~=2.4.0  
*  shapely~=2.1.2  
*  scikit-learn~=1.6.1  
*  tqdm~=4.67.1
